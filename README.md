


### 什么是云尚发卡系统？
专门为个人或小型企业提供在线售卡，订单处理系统

### 程序使用要求？
* php >= 5.6
* mysql >= 5.2
* win or linux

### 安装后务必添加nginx伪静态
```
location / {
   if (!-e $request_filename) {
   rewrite  ^(.*)$  /index.php?$1  last;
   break;
    }
 }

```
